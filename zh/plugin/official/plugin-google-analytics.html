# @vuepress/plugin-google-analytics (opens new window)

Google analytics 插件

# 安装

yarn add -D @vuepress/plugin-google-analytics
# OR npm install -D @vuepress/plugin-google-analytics

注意

如果你的项目正在使用 Google analytics 插件，推荐使用 Yarn (opens new window) 而不是 npm 来安装所有依赖。因为在这种情形下，npm 会生成错误的依赖树。

# 使用

module.exports = {
  plugins: [
    [
      '@vuepress/google-analytics',
      {
        'ga': '' // UA-00000000-0
      }
    ]
  ]
}

提示

请留意 GDPR (2018年欧盟数据保护规则改革) (opens new window), 在合适或者需要的情况下，考虑将 Google Analytics 设置为匿名化的 IP (opens new window)。

# 选项

# ga

  • 类型: string
  • 默认值: undefined

提供 Google Analytics ID 以启用集成。
